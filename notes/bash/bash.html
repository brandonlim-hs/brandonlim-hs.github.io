# .bash_profile vs .bashrc

so use .bash_profile on Mac

# Alias

  1. Create .bash_profile if not exist
touch ~/.bash_profile
  1. Append new alias to .bash_profile
echo "alias new_alias='some command'" >> .bash_profile
  1. Restart Terminal
  2. Verify new alias
alias new_alias

# Change Default Terminal Prompt

From:

192-168-1-104:~ brandonlim$ ls

To:

$ ls
  1. Open Terminal
  2. Create .bash_profile if not exist
touch ~/.bash_profile
  1. Append config to .bash_profile
echo "export PS1='$ '" >> .bash_profile
  1. Restart Terminal
